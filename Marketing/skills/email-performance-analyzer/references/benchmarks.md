# Email Industry Benchmarks Reference

Last updated: March 2026
Sources: Klaviyo (2024–2025), Litmus (2025), Mailchimp (2024), Campaign Monitor (2024)

---

## How to Use This File

Use these benchmarks ONLY when:
1. No brand baseline exists
2. No period-over-period data is available

Always state the source and caveat that benchmarks vary by list size, send frequency, and platform.

## Benchmark Caveats

Always include these caveats when using industry benchmarks:

1. **List size matters** — brands with <10K subscribers often see higher open/conversion rates than enterprise senders
2. **Send frequency matters** — brands sending 5+ times/week see lower per-email engagement than brands sending 1–2x/week
3. **MPP caveat** — open rate benchmarks vary significantly depending on whether the source uses MPP-inclusive or Non-MPP figures. Klaviyo 2025 benchmarks are MPP-inclusive unless noted
4. **Platform variation** — ESP platforms report metrics differently (unique vs. total opens, for example)
5. **Seasonality** — always note if the analysis period is in/out of peak season for the vertical
